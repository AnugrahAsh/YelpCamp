# YelpCamp
This is a CRUD Application used to get information about camping grounds 
You can add a camp
Edit it
Delete it
And Explore them
////////////////////////////////////////////////////
This app gives you info like pictures, price , location about the campgrounds
I used HTML, CSS, BootStrap, NodeJs, npm packages(such as ejs, methodoverride, path, etc), ExpressJs, MongoDB in this project
It is My first fully functional MERN App
